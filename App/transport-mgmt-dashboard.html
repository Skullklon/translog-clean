5 Mejoras Estratégicas (Bonus)
📍 Geolocalización de Rutas
Integrar Google Maps Directions API para trazar rutas reales, medir km exactos y calcular costos de combustible automáticamente. El QR check-in captura GPS del empleado para validar que estaba en el punto de abordaje correcto (radio de 200m).
🔀 Optimización de Rutas (IA)
Con los domicilios de empleados + direcciones de sucursales, usar Google OR-Tools o OSRM para sugerir agrupamiento óptimo de empleados por zona geográfica. Potencial ahorro de 15-30% en costo por redistribución de rutas ineficientes como R-04.
⛽ Costos Reales de Combustible
Webhook a API de precios de combustible local (ej. DGII en RD). Cada vehículo tiene registrado fuel_type y cost_per_km. El sistema actualiza automáticamente el costo base por ruta cuando varía el precio del combustible.
🚨 Detección de Fraude / Anomalías
Alertas automáticas por: check-ins QR desde ubicación diferente al punto de ruta, mismo empleado en 2 rutas simultáneas, costo por viaje fuera de ±2σ histórico, variación brusca en ocupación sin justificación, o firma OCR presente pero GPS ausente.
🚀 Estrategia de Escalado
Fase 1 — MVP (1-2 meses)Next.js + Supabase. QR check-in. OCR manual revisado. Dashboard básico. 1 proveedor, 3 sucursales.
Fase 2 — Escala (3-6 meses)Múltiples proveedores. Batch OCR automático. Reportes avanzados. API pública para integración ERP cliente.
Fase 3 — IA (6-12 meses)Optimización de rutas. Detección de fraude ML. Predicción de costos. App móvil nativa React Native.
